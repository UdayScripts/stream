# **App Name**: StreamGlide

## Core Features:

- Playlist URL Input: Users can input and submit an M3U playlist URL to load available channels.
- Dynamic Channel Listing: Parse and display a scrollable list of channels from the loaded M3U playlist, including channel names and logos (if provided).
- Integrated Video Player: Embedded video player to stream selected IPTV channels with essential controls like play/pause and volume.
- Channel Search & Filtering: Allow users to efficiently search for specific channels or filter them by categories derived from the playlist metadata.
- Saved Playlists: Persist previously entered M3U playlist URLs locally in the browser for quick re-access across sessions.
- AI Playlist Categorization Tool: A generative AI tool that automatically analyzes M3U playlist data to suggest and assign logical categories (e.g., News, Sports) to channels, enhancing navigation especially for sparse playlists.

## Style Guidelines:

- Background color: A deep charcoal gray with a subtle cool blue undertone (#202426), designed to provide a calm backdrop that allows video content and primary UI elements to shine.
- Primary interactive color: A vibrant yet calming digital blue (#66CCFF), evoking modernity and enhancing focus on content in a dark environment.
- Accent color: A fresh, energetic green (#47CC66), used sparingly for highlights, success states, and subtle interaction feedback.
- Unified font: 'Inter' (sans-serif), chosen for its modern, highly legible, and versatile nature, suitable for all textual elements from headlines to channel listings.
- Clean, minimalist line icons with consistent stroke weight, ensuring clarity and unobtrusive integration with the dark interface.
- A two-column adaptive design, featuring a prominent, immersive video playback area complemented by a collapsible channel list and search sidebar for efficient navigation across various screen sizes.
- Gentle, functional animations for UI state changes and navigation, such as smooth sidebar toggles and content loading indicators, to improve user flow and provide a polished feel.
